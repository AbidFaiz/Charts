## Usage :chart_with_upwards_trend:


<br/>
**Gradle**


- **Project level `build.gradle`**
```gradle
allprojects {
    repositories {
        maven { url 'https://jitpack.io' }
    }
}
```
- **App level `build.gradle`**
```gradle
dependencies {
    implementation 'com.github.AbidFaiz:ChartsProject:v3.0.3'
}
```
